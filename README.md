# xmucoder_nagi 的题解代码合集

本仓库整理了若干题目的参考实现（代码片段/完整程序）。大部分文件均可直接编译/运行；如遇到无法运行的情况，会在对应目录或文件中注明。

## 内容说明
### Part 1：
- `C语言/`：C 语言课程相关题目和积分赛题目
- `C++/`：C++ 课程项目，仅代码和题目
- `Java/`：课程实验相关代码，非工程化项目
- `Python/`：Python 小学期课程相关题目与练习

（此部分按照语言总结，目录较多，可使用 VS Code 的搜索/全局搜索快速定位题目关键词查找代码。）

## 来源与平台

- C 语言：主要来自厦大实验实训平台（[需校园网访问](10.26.57.15)）；部分来自 C 语言积分赛（[无需校园网](c语言积分赛.top)）
- C++ / Python：主要来自 xmuoj（[无需校园网](www.xmuoj.com)）
- 部分 C++ / Java：来自课程实验（非 OJ 题目）

## 运行参考

- C：建议使用 `MSVC` 编译运行, 通常情况 `gcc`/`clang`亦可
- C++：建议使用 `MSVC` 编译运行, 通常情况 `g++`/`clang++`亦可
- Java：使用 JDK（如 `javac`/`java`）运行，原环境 JDK 21.0.5
- Python：建议使用 Python 3 运行，原环境 Python 3.11

不同题目可能存在输入输出格式要求，请以题面为准。
